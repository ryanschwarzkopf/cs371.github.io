Linked Lists

A linked list is a linear collection of data that can exist in different parts of memory. A node has two parts: the first holds data, and the other hold a memory addressS. In the diagram every node is represented with a box around it. Data may only be accessed in order because you can only get to a node by visiting all of the previous nodes. Linked lists are useful for storing data of dynamic sizes as it allocates memory dynamically.
